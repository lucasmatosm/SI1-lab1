Sistemas de informção 1 UFCG

Uma experiência completa de desenvolvimento front-end

Porque escolher a disciplina?

O objetivo da disciplina é discutir técnicas de projeto de software. Haverá foco particular em técnicas utilizadas para melhorar o reuso e a flexibilidade de software desenvolvido usando Orientação a Objeto, mas discutiremos também projeto arquitetural. O contexto de todas as aplicações será a web. Assim, você também acabará a disciplina sabendo desenvolver sistemas web.

Funcionamento da disciplina

A disciplina funciona atravez de atividades continuas e individuais chamadas de laboratorios, onde o aluno podera exercitar os conteudos vistos em sala de aula, periodicamente serão realizados mini-testes surpresa e ao termino da disciplina sera realizado um projeto, onde os alunos poderão criar um sistema web bastante completo com os assuntos aprendidos ao longo do semestre.

Conclusão

Para a formação de um bom cientista da computacao é necessario que o mesmo, aprenda a desenvolver sistemas para desktop para aplicações mobile, e sistemas por web, por isso a disciplina de Sistemas de informação 1, é tao importante, ou seja, um dos 3 pilares de sustenção de um bom desenvolvedor de sistemas.
